## Dataset

El dataset utilizado en este proyecto está disponible en Hugging Face: [Dataset papas](https://huggingface.co/datasets/Carlos012/dataset_papas). Este conjunto de datos integra imágenes de tubérculos de papa obtenidas de repositorios en abiertos de internet y de capturas realizadas en un entorno controlado y local. Cada imagen cuenta con una etiqueta asociada a su categoría, lo que permite emplear el dataset en tareas de clasificación de tubérculos según su estado de conservación (cortes, signos de pudrición y brotes) durante la etapa de poscosecha.
