## Data

El dataset utilizado en este proyecto está disponible en Zenodo: [Dataset de tubérculos de papas](https://doi.org/10.5281/zenodo.20616991). Este conjunto de datos integra imágenes de tubérculos de papa obtenidas de repositorios públicos de internet y de fotografías realizadas en un entorno controlado y local. Cada imagen cuenta con una etiqueta asociada a su categoría (Buen estado o Defectuoso), lo que permite emplear el dataset en tareas de clasificación de tubérculos según su estado de conservación (cortes, signos de pudrición y brotes) durante la etapa de poscosecha.
